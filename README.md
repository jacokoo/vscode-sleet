# VS Code Sleet

[Sleet](https://github.com/JacoKoo/sleetjs) and [Sleet Handlebars](https://github.com/JacoKoo/sleet-handlebars)
plugin for [VS Code](code.visualstudio.com)

## Features

* Grammar highlight
* Compile on save

## Extension Commands


* `extension.sleet.compile`: compile current sleet file
